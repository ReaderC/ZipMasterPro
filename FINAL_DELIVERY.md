## 🚀 使用方法

### 方法1: 云编译 (推荐)
**无需苹果电脑！**

#### 使用GitHub Actions
1. **创建GitHub仓库**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/ZipMasterPro.git
   git push -u origin main
   ```

2. **触发自动构建**
   ```bash
   # 推送代码自动触发构建
   git add .
   git commit -m "Trigger build"
   git push
   ```

3. **下载IPA文件**
   - 进入GitHub Actions标签
   - 下载构建完成的IPA文件

#### 使用云编译脚本
```bash
# 运行云编译脚本
./cloud_build.sh

# 生成的IPA文件
# ZipMasterPro_unsigned.ipa
```

### 方法2: 本地编译
**需要苹果电脑**

#### 使用Xcode
1. **打开项目**
   ```bash
   open ZipMasterPro.xcodeproj
   ```

2. **连接设备**
   - 连接iPhone或iPad到电脑

3. **编译运行**
   - 点击运行按钮 (⌘R)
   - 应用自动安装到设备

#### 使用构建脚本
```bash
# 运行构建脚本
./build_ipa.sh

# 生成的IPA文件
# ipa/ZipMasterPro_unsigned.ipa
```
